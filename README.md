# Smart Student Performance Analytics System

## Overview
Smart Student Performance Analytics System is a Python-based application designed to manage and analyze student performance data efficiently. The system integrates Python with an SQL database to perform data storage, retrieval, and basic performance analysis.

## Features
- Add student records
- View student performance details
- Store data using SQLite database
- Menu-driven console application
- Clean and modular code structure

## Technologies Used
- Python
- SQLite (SQL Database)
- Git & GitHub
